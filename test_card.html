Energy Panel Card Test

This is a test page for the Energy Panel Card. In a real Home Assistant environment, the card would be loaded and would display energy data from your REST sensors.

Make sure you have set up the following:

  1. REST sensors in Home Assistant pointing to your energy API at http://100.107.164.25:9080/api/energy/data
  2. An input_datetime entity to select dates
  3. An automation to update the sensors when the date changes

Card Configuration:

type: custom:ha-energy-panel
title: Energy Consumption
power_entity: sensor.energy_power_data
gas_entity: sensor.energy_gas_data
dates_entity: sensor.energy_available_dates
date_input: input_datetime.energy_chart_date
